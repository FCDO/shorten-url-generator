# shorten-url-generator

## URL Shortener 短網址產生器

## 介紹
短網址產生器

## 功能
* 輸入網址轉換成短網址
* 點擊按鈕複製網址
* 透過短網址轉址到原本網址

## 開始使用

# Getting Start

## 1.請先確認有安裝 node.js 與 npm

## 2.將專案clone至本地

## 3.在本地開啟之後，透過終端機進入資料夾，輸入：
````````````
npm install
````````````

## 4.安裝完畢後，繼續輸入：

````````````
npm run dev
````````````

## 5.打開瀏覽器進入到以下網址

`````````````
http://localhost:3000
`````````````

## 6.若欲暫停使用

`````````````
ctrl+C
`````````````
